# auto_test_tbank-terminal

Скрипт для прохождения тестирования при активации терминала Т-Банк (Интернет-эквайринг).

## Установка

```bash
pip install -r requirements.txt
```

## Использование

```bash
python tbank_terminal_test.py
```

Скрипт запросит Terminal ID и пароль, затем покажет меню:

```
1 - Тест 1
2 - Тест 2
3 - Тест 3
7 - Тест 7 (с чеком)
8 - Тест 8
0 - Все тесты по порядку
q - Выход
```

## Описание тестов

| Тест | Действие |
|------|----------|
| 1 | Создание платежа |
| 2 | Создание платежа, оплата тестовой картой |
| 3 | Создание платежа, оплата, отмена |
| 7 | Создание платежа с чеком (Receipt) |
| 8 | Создание платежа, отмена (с повтором) |

## API

- [Init](https://developer.tbank.ru/eacq/api/init) - создание платежа
- [Cancel](https://developer.tbank.ru/eacq/api/cancel) - отмена платежа
- [Формирование токена](https://developer.tbank.ru/eacq/intro/developer/token)
